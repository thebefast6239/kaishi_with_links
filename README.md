## быстрая установка если уже стоит Kaishi 1.5k RU
1. скачайте файл 'Kaishi RU With Links.apkg'
<a href="https://github.com/thebefast6239/kaishi_with_links/raw/refs/heads/main/Kaishi%20RU%20With%20Links.apkg" target="_blank">(нажмите на этот текст чтобы скачать колоду напрямую)</a>
2. зайдите в Anki и импортируйте данную колоду
  на ПК: вверху кликнув 'Файл' -> 'Импортировать...' ( или нажав Ctrl+Shift+I )
  на android: справа сверху нажмите на три точки -> Импортировать -> Пакет колоды .apkg
3. затем удалите её, типы записей должны обновиться сами

## ручная установка
1. зайдите в Anki -> Инструменты -> Проверить медиафайлы -> Просмотреть файлы
  ( или откройте AppData\Roaming\Anki2\имявашегопрофиля\collection.media )
  перенесите в открывшуюся папку иконки из папки 'media' которая прикреплена здесь
2. зайдите в Anki, Ctrl+Shift+N, выберите существующий тип записи (если уже стоит колода Kaishi, это будет 'Kaishi 1.5k RU'
3. нажмите на "Карточки"
4. выберите "Шаблон лица" и вставьте туда весь код из 'front.html'
5. выберите "Шаблон оборота" и вставьте туда весь код из 'back.html'
6. выберите "Таблица стилей" и вставьте туда весь код из 'styles.css'
7. сохраните
