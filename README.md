# kaishi_with_links
kaishi improved html/css
![Пример карточки](https://github.com/thebefast6239/kaishi_with_links/raw/main/example.jpg)

## быстрая установка если уже стоит Kaishi 1.5k RU
1. скачайте 'Kaishi RU With Links.apkg'
2. импортируйте данную колоду в Anki, затем удалите её, типы записей должны обновиться сами

## ручная установка
1. зайдите в Anki -> Инструменты -> Проверить медиафайлы -> Просмотреть файлы
  ( или откройте AppData\Roaming\Anki2\имявашегопрофиля\collection.media )
  перенесите в открывшуюся папку иконки из папки 'media' которая прикреплена здесь
2. зайдите в Anki, Ctrl+Shift+N, выберите существующий тип записи (если уже стоит колода Kaishi, это будет 'Kaishi 1.5k RU'
3. нажмите на "Карточки"
4. выберите "Шаблон оборота" и вставьте туда весь код из 'back.html'
5. выберите "Таблица стилей" и вставьте туда весь код из 'styles.css'
6. сохраните
